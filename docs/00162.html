Library Interface
a) Initialization Function
 
Name 
Description 
 
ADCx_Initialize 
Initializes given instance of ADC peripheral. 
b) Setup Functions
 
Name 
Description 
 
ADC_ChannelSamplingStop 
Stops sampling on the ADC core and channel that was specified using the ADC_SoftwareControlledConversionEnable() API 
 
ADC_CompareEnable 
Enables digital comparisons for the given channel on the specified ADC core 
 
ADC_CompareDisable 
Disables digital comparisons for the given channel on the specified ADC core 
 
ADC_CompareWinThresholdSet 
Configures the low and high threshold values for the comparator on the specified ADC core 
 
ADC_CoreInterruptsDisable 
Disable the specified ADC core's interrupts 
 
ADC_CoreInterruptsEnable 
Enables the specified ADC core's interrupt 
 
ADC_CoreInterruptsStatusClear 
Clears the specified ADC core's interrupt status 
 
ADC_COREnCallbackRegister 
Registers the ADC core specific callback function 
 
ADC_CompareWinModeSet 
Configures the low and high threshold values for the comparator on the specified ADC core 
 
ADC_Enable 
Enables the ADC module 
 
ADC_SyncTriggerEnable 
Enables sync trigger defined by CTRLC.CNT value 
 
ADC_GlobalLevelConversionStop 
Stops global level trigger and stops conversion on channels configured with Global level as the trigger source 
 
ADC_SyncTriggerCounterSet 
Configures the value of CTRLC.CNT register 
 
ADC_SyncTriggerDisable 
Disables sync trigger 
 
ADC_Disable 
Disables the ADC module 
 
ADC_GlobalCallbackRegister 
Registers the global callback function 
 
ADC_ChannelSamplingStart 
Enables the Analog Mux Input and starts sampling on the ADC core and channel that was specified using the ADC_SoftwareControlledConversionEnable() API 
 
ADC_SoftwareControlledConversionEnable 
Enables software controlled conversion on the specified ADC core's channel 
c) Transaction Functions
 
Name 
Description 
 
ADC_FIFOBufferRead 
Returns multiple ADC conversion values from the ADC FIFO 
 
ADC_FIFORead 
Returns a single ADC conversion value from the ADC FIFO 
 
ADC_ResultGet 
Returns the ADC conversion value for the specified ADC core channel 
 
ADC_GlobalLevelConversionStart 
Starts global level trigger and starts conversion on channels configured with Global level as the trigger source 
 
ADC_GlobalEdgeConversionStart 
Generates global edge trigger and starts conversion on channels configured with Global edge as the trigger source 
 
ADC_ChannelConversionStart 
Starts conversion on the ADC core and channel that was specified using the ADC_SoftwareControlledConversionEnable() API 
d) Status Functions
 
Name 
Description 
 
ADC_ChannelResultIsReady 
Indicates whether ADC conversion is complete on the specified ADC core's channel 
 
ADC_CoreInterruptsStatusGet 
Returns the specified ADC core's interrupt status 
 
ADC_EOSStatusGet 
Indicates whether End of scan interrupt flag is set on the specified ADC core 
 
ADC_GlobalInterruptsStatusGet 
Returns the global interrupt status
