Library Interface
a) Initialization Function
 
Name 
Description 
 
FLEXCOMx_USART_Initialize 
Initializes given instance of the FLEXCOM USART peripheral. 
b) Setup Functions
 
Name 
Description 
 
FLEXCOMx_USART_SerialSetup 
Sets the FLEXCOM USART serial communication settings dynamically. 
 
FLEXCOMx_USART_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given FLEXCOM USART's read events occur. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given FLEXCOM USART's write events occur. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given FLEXCOM_USART's read events occur. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given FLEXCOM_USART's write events occur. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_ReadNotificationEnable 
This API lets the application turn the receive notifications on/off. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_WriteNotificationEnable 
This API lets the application turn the transmit notifications on/off. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_ReadThresholdSet 
This API allows the application to set a threshold level on the number of bytes of data available in the receive buffer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_WriteThresholdSet 
This API allows the application to set a threshold level on the number of free space available in the transmit buffer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
c) Transaction Functions
 
Name 
Description 
 
FLEXCOMx_USART_Read 
Submits a read buffer to the given FLEXCOM USART peripheral to process. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_ReadByte 
Submits request to read a byte of data to the given FLEXCOM USART peripheral. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_Write 
Submits a write buffer to the given FLEXCOM USART peripheral to transfer. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_WriteByte 
Submits a byte of data to the given FLEXCOM USART peripheral to transfer. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_Read 
Read the requested data from the receive buffer into the application buffer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_Write 
Submits a write buffer to the given FLEXCOM_USART peripheral to transfer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
d) Status Functions
 
Name 
Description 
 
FLEXCOMx_USART_ErrorGet 
Gets the error of the given FLEXCOM USART peripheral instance. 
 
FLEXCOMx_USART_ReadCountGet 
Returns the count of number of bytes processed for a given FLEXCOM USART read operation. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_ReadIsBusy 
Returns the read request status associated with the given FLEXCOM USART peripheral instance. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_ReceiverIsReady 
Returns the hardware status of the FLEXCOM USART Receiver. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_TransmitterIsReady 
Returns the hardware status of the FLEXCOM USART Transmitter. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_WriteCountGet 
Returns the count of number of bytes processed for a given FLEXCOM USART write operation. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_WriteIsBusy 
Returns the write request status associated with the given FLEXCOM USART peripheral instance. This API is available when USART ring buffer mode is disabled in MHC. 
 
FLEXCOMx_USART_ReadBufferSizeGet 
Returns the size of the receive ring buffer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_WriteBufferSizeGet 
Returns the size of the internal transmit ring buffer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_ReadCountGet 
Returns the number of bytes available in the receive buffer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_WriteCountGet 
Returns the number of bytes pending to be transmitted out in the transmit buffer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_ReadFreeBufferCountGet 
Returns the number of bytes of free space available in the internal receive buffer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
 
FLEXCOMx_USART_WriteFreeBufferCountGet 
Returns the number of bytes of free space available in the internal transmit buffer. This API is only available when FLEXCOM_USART ring buffer mode is enabled in MHC. 
e) Data Types and Constants
 
Name 
Description 
 
FLEXCOM_USART_RING_BUFFER_CALLBACK 
Defines the data type and function signature for the FLEXCOM_USART peripheral callback function in the ring buffer mode. 
 
FLEXCOM_USART_CALLBACK 
Defines the data type and function signature for the FLEXCOM USART peripheral callback function. 
 
FLEXCOM_USART_DATA 
Defines the data type for the FLEXCOM USART serial setup. 
 
FLEXCOM_USART_ERROR 
Defines the data type for the FLEXCOM USART peripheral errors. 
 
FLEXCOM_USART_PARITY 
This is type flexcom_11268_usart::FLEXCOM_USART_PARITY. 
 
FLEXCOM_USART_SERIAL_SETUP 
This is type flexcom_11268_usart::FLEXCOM_USART_SERIAL_SETUP. 
 
FLEXCOM_USART_STOP 
This is type flexcom_11268_usart::FLEXCOM_USART_STOP. 
 
FLEXCOM_USART_EVENT 
Defines the data type for the FLEXCOM USART events.
