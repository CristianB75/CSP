FCW_PFM_WriteProtectRegionSetup Function

This function creates a program flash region of given size and address. it later can be write protected using FCW_PFM_WriteProtectEnable API.

C
void FCW_PFM_WriteProtectRegionSetup(
    PFM_WP_REGION region, 
    PFM_WP_REGION_SETUP setupStruct
);
Preconditions

None

Parameters
Parameters 
Description 
region 
Region number 
setupStruct 
structure having detail of the region like base address, size etc 
Returns

None.

Example
    PFM_WP_REGION_SETUP setupStruct;
    setupStruct.regionBaseAddress = 0xC000000U
    setupStruct.mirrorEnable = true;
    setupStruct.regionSize = 0 //4KB region
    FCW_PFM_WriteProtectRegionSetup(PFM_WP_REGION_2, setupStruct);
