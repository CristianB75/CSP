Library Interface
a) Initialization Function
 
Name 
Description 
 
CANx_Initialize 
Initializes given instance of the CAN peripheral. 
b) Setup Functions
 
Name 
Description 
 
CANx_TxBuffersCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given CAN's Tx transfer events occur. 
 
CANx_TxFifoCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given CAN's Tx transfer events occur. 
 
CANx_TxEventFifoCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given CAN's Tx transfer events occur. 
 
CANx_RxBuffersCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given CAN's Rx transfer events occur. 
 
CANx_RxFifoCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given CAN's Rx transfer events occur. 
c) Transaction Functions
 
Name 
Description 
 
CANx_ErrorGet 
Returns the error during transfer. 
 
CANx_MessageTransmit 
Transmits a message into CAN bus from the specific Tx buffer. 
 
CANx_InterruptClear 
Clears Interrupt status. 
 
CANx_TxBufferIsBusy 
Check if Transmission request is pending for the specific Tx buffer. 
 
CANx_InterruptGet 
Returns the Interrupt status. 
 
CANx_MessageTransmitFifo 
Transmit multiple messages into CAN bus from Tx FIFO. 
 
CANx_MessageRAMConfigSet 
Set the Message RAM Configuration. 
 
CANx_TxFifoFreeLevelGet 
Returns Tx FIFO Free Level. 
 
CANx_TxEventFifoRead 
Read Tx Event FIFO for the transmitted messages. 
 
CANx_ExtendedFilterElementGet 
Get a Extended filter element configuration. 
 
CANx_TxEventFifoFillLevelGet 
Returns Tx Event FIFO Fill Level. 
 
CANx_ExtendedFilterElementSet 
Set a Extended filter element configuration. 
 
CANx_MessageReceive 
Read a message from the specific Rx Buffer. 
 
CANx_RxBufferNumberGet 
Get Rx Buffer Number. 
 
CANx_MessageReceiveFifo 
Read messages from Rx FIFO0/FIFO1. 
 
CANx_StandardFilterElementGet 
Get a standard filter element configuration. 
 
CANx_RxFifoFillLevelGet 
Returns Rx FIFO0/FIFO1 Fill Level. 
 
CANx_StandardFilterElementSet 
Set a standard filter element configuration. 
 
CANx_SleepModeEnter 
Puts the CAN Peripheral in sleep mode (clock stop request). 
 
CANx_SleepModeExit 
Exits CAN peripheral from sleep mode (clock stop request). 
d) Status Functions
 
Name 
Description 
 
CANx_ErrorCountGet 
Returns the transmit and receive error count during transfer.
