HEMC_HeccGetStatus Function

Get the HECC status of the HEMC peripheral.

C
HEMC_HECC_STATUS HEMC_HeccGetStatus();
Preconditions

None.

Returns

Current status of the HEMC peripheral.

Remarks

None

Example
  HEMC_HECC_STATUS status;
  status = HEMC_HeccGetStatus();
