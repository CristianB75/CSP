# ADCHS\_GlobalLevelConversionStop Function

**Parent topic:**[Analog to Digital Converter \(ADCHS\)](GUID-8740EC52-3365-4B31-B19A-227EC55268DD.md)

## C

```c
void ADCHS_GlobalLevelConversionStop(void)
```

## Summary

Stops the global software level trigger

## Description

This function resets the global level software trigger bit. This stops ADC conversion of analog channels<br />configured for global software level trigger.

## Precondition

ADCHS\_Initialize\(\) function must have been called first for the associated instance and ADC modules corresponding to channels must have been enabled using ADCHS\_ModulesEnable\(\) function.

## Parameters

None.

## Returns

None.

## Example

```c
ADCHS_Initialize();
ADCHS_ModulelsEnable(ADCHS_MODULE0_MASK);
ADCHS_GlobalLevelConversionStop();
```

## Remarks

None
