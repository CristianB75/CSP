FLEXCOM_TWI_SLAVE_TRANSFER_EVENT Enumeration

FLEXCOM TWI Slave Transfer Event Enums. 

Defines the enum for the TWI slave transfer event. An event of this type is returned back in the callback by the FLEXCOM slave PLIB to indicate the transfer event for which the callback has been called.

C
typedef enum {
  FLEXCOM_TWI_SLAVE_TRANSFER_EVENT_NONE,
  FLEXCOM_TWI_SLAVE_TRANSFER_EVENT_ADDR_MATCH,
  FLEXCOM_TWI_SLAVE_TRANSFER_EVENT_RX_READY,
  FLEXCOM_TWI_SLAVE_TRANSFER_EVENT_TX_READY,
  FLEXCOM_TWI_SLAVE_TRANSFER_EVENT_TRANSMISSION_COMPLETE
} FLEXCOM_TWI_SLAVE_TRANSFER_EVENT;
Members
Members 
Description 
FLEXCOM_TWI_SLAVE_TRANSFER_EVENT_ADDR_MATCH 
Address match event 
FLEXCOM_TWI_SLAVE_TRANSFER_EVENT_RX_READY 
Data sent by TWI Master is available 
FLEXCOM_TWI_SLAVE_TRANSFER_EVENT_TX_READY 
TWI slave can respond to data read request from TWI Master 
FLEXCOM_TWI_SLAVE_TRANSFER_EVENT_TRANSMISSION_COMPLETE 
TWI stop condition received or start condtion with other slave address detected 
Remarks

All the event flags may not be available on FLEXCOM of the selected device. Refer the header file of the generated code for the event flags available on the device.
