Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
ADCHSx_Initialize 
Initializes given instance of ADCHS peripheral. 
b) Setup Functions
 
Name 
Description 
 
ADCHSx_CallbackRegister 
Registers the function to be called from interrupt 
 
ADCHSx_EOSCallbackRegister 
Registers the function to be called from end of scan interrupt 
 
ADCHSx_ComparatorCallbackRegister 
Registers the function to be called from the ADC comparator interrupt 
 
ADCHSx_ComparatorDisable 
Enables comparator 
 
ADCHSx_ComparatorEnable 
Enables comparator 
 
ADCHSx_ComparatorEventModeSet 
Sets the condition under which the comparator output becomes true 
 
ADCHSx_ComparatorLimitSet 
Sets lower and higher threshold for comparator 
 
ADCHSx_DMACallbackRegister 
Registers the application event handler which is called when the DMA transfer is complete 
 
ADCHSx_DMAResultBaseAddrSet 
Registers the user-defined RAM address at which the DMA engine will start saving the converted data 
 
ADCHSx_DMASampleCountBaseAddrSet 
Registers the user-defined RAM address at which the DMA engine will start saving the current count of output samples 
 
ADCHSx_FilterCallbackRegister 
Registers the function to be called from the ADC filter ready interrupt 
c) Transaction Functions
 
Name 
Description 
 
ADCHSx_ChannelConversionStart 
Starts the ADC conversion of an analog channel 
 
ADCHSx_ChannelEarlyInterruptDisable 
Disables the ADC channel early result data interrupt 
 
ADCHSx_ChannelEarlyInterruptEnable 
Enables the ADC channel early result data interrupt 
 
ADCHSx_ChannelResultGet 
Reads the conversion result of the channel 
 
ADCHSx_ChannelResultInterruptDisable 
Disables the ADC channel result data interrupt 
 
ADCHSx_ChannelResultInterruptEnable 
Enables the ADC channel result data interrupt 
 
ADCHSx_GlobalEdgeConversionStart 
Starts the ADC conversion of analog channels configured for global software edge trigger 
 
ADCHSx_GlobalLevelConversionStart 
Starts the ADC conversion of analog channels configured for global software level trigger 
 
ADCHSx_GlobalLevelConversionStop 
Stops the global software level trigger 
 
ADCHSx_ModulesDisable 
Disables the ADC modules 
 
ADCHSx_ModulesEnable 
Enables the ADC modules 
 
ADCHSx_FilterDataGet 
Returns the ADC filter data 
d) Status Functions
 
Name 
Description 
 
ADCHSx_ChannelResultIsReady 
Returns the status of the channel conversion 
 
ADCHSx_EOSStatusGet 
Returns the status of the end of scan interrupt flag 
 
ADCHSx_ComparatorAnalogInputIDGet 
Identifies the analog input for which the comparator event is generated 
 
ADCHSx_DMAStatusGet 
Returns the DMA transfer status 
 
ADCHSx_FilterIsReady 
Returns the status of ADC filter 
e) Data Types and Constants
 
Name 
Description 
 
ADCHS_CALLBACK 
Defines the function pointer data type and function signature for the adchs peripheral callback function. 
 
ADCHS_CALLBACK_OBJECT 
Callback structure 
 
ADCHS_CHANNEL_NUM 
Identifies ADCHS inputs as analog channel number 
 
ADCHS_MODULE_MASK 
Identifies dedicated and shared ADC module mask 
 
ADCHS_EOS_CALLBACK 
Defines the function pointer data type and function signature for the adchs end of scan interrupt callback function. 
 
ADCHS_EOS_CALLBACK_OBJECT 
Callback structure for End of Scan (EOS) interrupt 
 
ADCHS_DC_CALLBACK 
Defines the function pointer data type and function signature for the adchs digital comparator interrupt callback function. 
 
ADCHS_DF_CALLBACK 
Defines the function pointer data type and function signature for the adchs digital filter interrupt callback function. 
 
ADCHS_DMA_CALLBACK 
Defines the function pointer data type and function signature for the adchs DMA interrupt callback function. 
 
ADCHS_CMP_EVENT_MODE 
Identifies ADCHS comparator event modes 
 
ADCHS_DMA_STATUS 
Identifies ADCHS DMA status
