Running The Application

MPU Devices

  1. Build the IAR or MPLAB X project using IAR Embedded Workbench or MPLAB X IDE respectively
  2. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  3. Reset the board to run the application

 

Kit Name 
Output 
SAMA5D2 Xplained Ultra evaluation kit 

  • RGB_LED toggles in Blue color indicating the board is up and running
  • Press switch PB_USER to put the board into shutdown mode
  • Short the PIN 1 of Connector J12 to 3V3 supply
  • Board reboots and executes the previous application (indicated by the RGB_LED toggling in blue color)

 

SAM9X60 evaluation kit 

  • RGB_LED toggles in Blue color indicating the board is up and running
  • Press switch SW1 to put the board into shutdown mode
  • Press switch SW2 to wake up processor
  • Board reboots and executes the previous application (indicated by the RGB_LED toggling in blue color)

 

 

Note:

  1. SAM9X60 evaluation kit - 3.3V battery (supercapacitor) is implemented to permanently maintain the VDDBU voltage. This function allows the user to shut down the MPU and the system, thus entering a low power mode, and still keep the custom configuration that was previously set in the MPU backup area. While in Shut-down mode, the board can be woken up by action on the SW2 button (WAKE UP), which signals the MPU to resume operations. Jumper JP1/J2 must be in place for proper operation of the MPU, and can be removed if the user wants to bring the MPU back to the initial configuration, by resetting the General Purpose Backup Registers (GPBR).
  2. Execute the following steps to bring the MPU back to the initial configuration in the SAM9X60 evaluation kit
    • Power off the SAM9X60 Evaluation kit board and then remove jumper J2 (VDDBU) (open)
    • Power on the SAM9X60 Evaluation kit board for 10 seconds
    • Power off the SAM9X60 Evaluation kit board and then put the jumper back to J2 (VDDBU) (close)
