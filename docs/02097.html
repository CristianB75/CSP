CRC_CRCCalculate Function

It first configures the CRC engine data Width to 8-bit mode and then the initial seed value passed as argument. The user data from buffer will then be written in CRCDAT register. 

This function returns the final CRC value once the computation is done 

This function can be called in repeatedly for multiple blocks of memory/data by using previously generated CRC value as seed.

C
uint32_t CRC_CRCCalculate(
    void * buffer, 
    uint32_t length, 
    uint32_t seed
);
Parameters
Parameters 
Description 
buffer 
Buffer on which the CRC has to be computed
 
length 
Size of the buffer in bytes.
 
seed 
CRC initial value (seed). It should be in direct form 
Returns

crc - Generated crc value

Remarks

None

Example
#define DMAC_TRANSFER_SIZE   9

const uint8_t srcBuffer[DMAC_TRANSFER_SIZE] = {'1', '2', '3', '4', '5', '6', '7', '8', '9'};

CRC_SETUP crcSetup = {0};

crcSetup.reverse_crc_input  = true;
crcSetup.polynomial_length  = 32;
crcSetup.polynomial         = 0x04C11DB7;
crcSetup.reverse_crc_output = true;
crcSetup.final_xor_value    = 0xFFFFFFFF;

CRC_CRCSetup(crcSetup);

crc_32 = DMAC_CRCCalculate((void *)srcBuffer, DMAC_TRANSFER_SIZE, 0xFFFFFFFF);
