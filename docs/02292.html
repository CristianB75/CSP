Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
DMA_Initialize 
Initializes the DMA controller of the device. 
b) Setup Functions
 
Name 
Description 
 
DMA_ChannelCallbackRegister 
This function allows a DMA PLIB client to set an event handler. 
 
DMA_ChannelDisable 
The function disables the specified DMA channel. 
 
DMA_ChannelEnable 
The function enables the specified DMA channel. 
 
DMA_ChannelInterruptDisable 
This API is used to disable DMA channel interrupt 
 
DMA_ChannelInterruptEnable 
This API is used to enable DMA channel interrupt 
 
DMA_ChannelLinkedListTransfer 
The function submits a list of DMA transfers. 
 
DMA_ChannelPatternIgnoreByteDisable 
Disable DMA channel pattern ignore byte 
 
DMA_ChannelPatternIgnoreByteEnable 
Enable DMA channel pattern ignore byte 
 
DMA_ChannelPatternIgnoreValue 
Configures the byte to be ignored for pattern matching when it is enabled 
 
DMA_ChannelPatternMatchDisable 
Disable DMA channel pattern matching 
 
DMA_ChannelPatternMatchEnable 
Enable DMA channel pattern matching 
 
DMA_ChannelPatternMatchSetup 
This API is used to setup DMA pattern matching. 
 
DMA_ChannelSettingsGet 
Returns the current channel settings for the specified DMA Channel 
 
DMA_ChannelSettingsSet 
Changes the current transfer settings of the specified DMA channel. 
c) Transaction Functions
 
Name 
Description 
 
DMA_ChannelTransfer 
Schedules a DMA transfer on the specified DMA channel. 
d) Status Functions
 
Name 
Description 
 
DMA_ChannelGetTransferredCount 
Returns transfer count of the ongoing DMA trasnfer 
 
DMA_ChannelInterruptFlagsGet 
This API is used to get the DMA channel interrupt flags 
 
DMA_ChannelIsBusy 
The function returns the busy status of the channel. 
e) Data Types and Constants
 
Name 
Description 
 
DMA_CHANNEL 
Lists the set of channels available for data transfer using DMA. 
 
DMA_CHANNEL_CALLBACK 
Pointer to a DMA Transfer Event handler function. 
 
DMA_CHANNEL_CONFIG 
Defines the data type for DMA channel configuration 
 
DMA_PATTERN_MATCH_LEN 
Enumeration for DMA pattern match length type 
 
DMA_TRANSFER_EVENT 
Defines the data type for DMA_TRANSFER_EVENT 
 
DMA_INT_BLOCK_HALF_TRANSFER_COMPLETE 
Macro for Half Block transfer complete interrupt 
 
DMA_INT_BLOCK_TRANSFER_COMPLETE 
Macro for Block transfer complete interrupt 
 
DMA_INT_CELL_TRANSFER_COMPLETE 
Macro for cell transfer complete interrupt 
 
DMA_INT_LINKED_LIST_DONE 
Macro for Linked list transfer complete interrupt 
 
DMA_INT_READ_ERROR 
Macro for read error interrupt 
 
DMA_INT_START_DETECTED 
Macro for start detected interrupt 
 
DMA_INT_TRANSFER_ABORT 
Macro for transfer abort interrupt 
 
DMA_INT_WRITE_ERROR 
Macro for write error interrupt 
 
DMA_TRANSFER_EVENT_BLOCK_TRANSFER_COMPLETE 
Block was transferred successfully. 
 
DMA_TRANSFER_EVENT_CELL_TRANSFER_COMPLETE 
Macro for DMA cell transfer complete event 
 
DMA_TRANSFER_EVENT_ERROR 
Error while processing the request 
 
DMA_TRANSFER_EVENT_HALF_BLOCK_TRANSFER_COMPLETE 
Macro for half block transfer complete event 
 
DMA_TRANSFER_EVENT_LINKED_LIST_TRANSFER_COMPLETE 
Macro for linked list transfer complete event 
 
DMA_TRANSFER_EVENT_START_DETECTED 
Macro for DMA start trigger detected and block transfer has started 
 
DMA_TRANSFER_EVENT_TRANSFER_ABORTED 
Macro for DMA abort trigger detected and block transfer has been aborted 
 
DMA_DESCRIPTOR_REGS 
DMA linked list descriptor
