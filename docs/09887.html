Library Interface
a) Initialization Function
 
Name 
Description 
 
UARTx_Initialize 
Initializes given instance of the UART peripheral. 
b) Setup Functions
 
Name 
Description 
 
UARTx_SerialSetup 
Sets the UART serial communication settings dynamically. 
 
UARTx_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given UART's read events occur. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given UART's write events occur. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given UART's read events occur. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given UART's write events occur. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_ReadNotificationEnable 
This API lets the application turn the receive notifications on/off. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_WriteNotificationEnable 
This API lets the application turn the transmit notifications on/off. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_ReadThresholdSet 
This API allows the application to set a threshold level on the number of bytes of data available in the receive buffer. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_WriteThresholdSet 
This API allows the application to set a threshold level on the number of free space available in the transmit buffer. This API is only available when UART ring buffer mode is enabled in MHC. 
c) Transaction Functions
 
Name 
Description 
 
UARTx_Read 
Submits request to read n-Bytes of data to the given UART peripheral. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_ReadByte 
Submits request to read a byte of data to the given UART peripheral. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_Write 
Submits a write buffer to the given UART peripheral to transfer. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_WriteByte 
Submits a byte of data to the given UART peripheral to transfer. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_Read 
Read the requested data from the receive buffer into the application buffer. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_Write 
Submits a write buffer to the given UART peripheral to transfer. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_ReadAbort 
Aborts the ongoing read request. This API is available when UART ring buffer mode is disabled in MHC. 
d) Status Functions
 
Name 
Description 
 
UARTx_ErrorGet 
Gets the error of the given UART peripheral instance. 
 
UARTx_ReadCountGet 
Returns the count of number of bytes processed for a given UART read operation. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_WriteCountGet 
Returns the count of number of bytes processed for a given UART write operation. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_ReadIsBusy 
Returns the read request status associated with the given UART peripheral instance. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_WriteIsBusy 
Returns the write request status associated with the given UART peripheral instance. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_ReceiverIsReady 
Returns the hardware status of the UART Receiver. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_TransmitterIsReady 
Returns the hardware status of the UART Transmitter. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_TransmitComplete 
Returns the hardware status of the UART Transmit Shift Register. This API is available when UART ring buffer mode is disabled in MHC. 
 
UARTx_ReadBufferSizeGet 
Returns the size of the receive ring buffer. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_WriteBufferSizeGet 
Returns the size of the internal transmit ring buffer. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_ReadCountGet 
Returns the number of bytes available in the receive buffer. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_WriteCountGet 
Returns the number of bytes pending to be transmitted out in the transmit buffer. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_ReadFreeBufferCountGet 
Returns the number of bytes of free space available in the internal receive buffer. This API is only available when UART ring buffer mode is enabled in MHC. 
 
UARTx_WriteFreeBufferCountGet 
Returns the number of bytes of free space available in the internal transmit buffer. This API is only available when UART ring buffer mode is enabled in MHC. 
e) Data Types and Constants
 
Name 
Description 
 
UART_CALLBACK 
Defines the data type and function signature for the UART peripheral callback function. 
 
UART_RING_BUFFER_CALLBACK 
Defines the data type and function signature for the UART peripheral callback function in the ring buffer mode. 
 
UART_EVENT 
Defines the data type for the UART events. 
 
UART_ERROR 
Defines the data type for the UART peripheral errors. 
 
UART_PARITY 
Defines the type of parity for UART peripheral. 
 
UART_SERIAL_SETUP 
Defines the data type for the UART serial setup.
