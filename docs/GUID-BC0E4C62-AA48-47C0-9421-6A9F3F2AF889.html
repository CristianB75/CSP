void dcache_CleanByAddr (uint32_t *addr, uint32_t size)

Summary

Clean Data Cache by address.

Description

Clean L1 Data Cache by address starting from address to address+size.

Precondition

None.

Parameters

Param Description
addr Source buffer address
size buffer length in bytes

Returns

None.
