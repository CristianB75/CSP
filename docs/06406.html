Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
SERCOMx_I2C_Initialize 
Initializes the instance of the SERCOM peripheral operating in I2C mode. 
b) Setup Functions
 
Name 
Description 
 
SERCOMx_I2C_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given SERCOM I2C's transfer events occur. 
c) Transaction Functions
 
Name 
Description 
 
SERCOMx_I2C_ReadByte 
Read the received I2C byte 
 
SERCOMx_I2C_WriteByte 
Write a data byte to I2C master 
 
SERCOMx_I2C_AckActionSet 
Sets the ACK action for the next byte that will be received from the I2C master. 
 
SERCOMx_I2C_CommandSet 
Sets I2C slave command. 
d) Status Functions
 
Name 
Description 
 
SERCOMx_I2C_TransferDirGet 
Returns the I2C transfer direction. 
 
SERCOMx_I2C_LastByteAckStatusGet 
Returns the ACK status of the last byte written to the I2C master. 
 
SERCOMx_I2C_InterruptFlagsGet 
Returns the SERCOM I2C slave interrupt flags. 
 
SERCOMx_I2C_InterruptFlagsClear 
Clears the specified SERCOM I2C slave interrupt flags. 
 
SERCOMx_I2C_ErrorGet 
Returns the I2C error occurred on the bus. 
 
SERCOMx_I2C_IsBusy 
Returns the Peripheral busy status. 
e) Data Types and Constants
 
Name 
Description 
 
SERCOM_I2C_SLAVE_ACK_ACTION_SEND 
Defines the enum for the I2C slave acknowledgement type. 
 
SERCOM_I2C_SLAVE_ACK_STATUS 
Defines the enum for the I2C slave acknowledgement. 
 
SERCOM_I2C_SLAVE_CALLBACK 
Defines the data type and function signature for the SERCOM I2C Slave callback function. 
 
SERCOM_I2C_SLAVE_COMMAND 
Defines the enum for the I2C slave commands. 
 
SERCOM_I2C_SLAVE_ERROR 
Defines the enum for the I2C slave error. 
 
SERCOM_I2C_SLAVE_INTFLAG 
Defines the enum for the I2C slave interrupt flags. 
 
SERCOM_I2C_SLAVE_TRANSFER_DIR 
Defines the enum for I2C data transfer direction. 
 
SERCOM_I2C_SLAVE_TRANSFER_EVENT 
Defines the enum for the I2C slave transfer event.
