Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
CANx_Initialize 
Initializes given instance of the CAN peripheral. 
b) Setup Functions
 
Name 
Description 
 
CANx_RxCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given CAN's Rx transfer events occur. 
 
CANx_TxCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given CAN's Tx transfer events occur. 
c) Transaction Functions
 
Name 
Description 
 
CANx_InterruptDisable 
Disables CAN Interrupt. 
 
CANx_InterruptEnable 
Enables CAN Interrupt. 
 
CANx_InterruptGet 
Returns the Interrupt status. 
 
CANx_MessageAbort 
Abort request for a Mailbox. 
 
CANx_MessageAcceptanceMaskGet 
Get Message acceptance identifier mask from mailbox. 
 
CANx_MessageAcceptanceMaskSet 
Set Message acceptance identifier mask in mailbox. 
 
CANx_MessageIDGet 
Get Message ID from mailbox. 
 
CANx_MessageIDSet 
Set Message ID in mailbox. 
 
CANx_MessageTimestampGet 
Get the message timestamp from a mailbox. 
 
CANx_MessageTransmit 
Transmits a message into CAN bus. 
 
CANx_MessageReceive 
Receives a message from CAN bus. 
d) Status Functions
 
Name 
Description 
 
CANx_ErrorGet 
Returns the error during transfer. 
 
CANx_ErrorCountGet 
Returns the transmit and receive error count during transfer. 
 
CANx_MailboxIsReady 
Returns true if Mailbox is ready otherwise false. 
e) Data Types and Constants
 
Name 
Description 
 
CAN_CALLBACK 
CAN Callback Function Pointer. 
 
CAN_ERROR 
CAN Transfer Error data type. 
 
CAN_INTERRUPT_MASK 
CAN Interrupt Mask. 
 
CAN_MAILBOX_MASK 
CAN Mailbox Mask. 
 
CAN_MAILBOX_NUM 
CAN Mailbox Number. 
 
CAN_MAILBOX_RX_ATTRIBUTE 
CAN Mailbox RX Attribute for Reception Mailbox, Reception Mailbox with overwrite, Consumer Rx Mailbox and Producer Rx Mailbox. 
 
CAN_MAILBOX_STATE 
CAN Mailbox State. 
 
CAN_MAILBOX_TX_ATTRIBUTE 
CAN Mailbox TX Attribute for Transmit mailbox, Producer Tx Mailbox and Consumer Tx Mailbox. 
 
CAN_RX_MSG 
CAN PLib Object structure. 
 
CAN_MAILBOX_CALLBACK 
CAN transfer event callback structure. 
 
CAN_MSG_RX_ATTRIBUTE 
CAN Message RX Attribute for Data Frame and Remote Frame.
