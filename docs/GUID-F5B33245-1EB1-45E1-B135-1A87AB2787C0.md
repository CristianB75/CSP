```c
void DWT_Initialize(void)
```

## Summary

Initializes Data Watchpoint and Trace \(DWT\) unit.

## Description

This function initializes the Data Watchpoint and Trace \(DWT\) unit as configured<br />by the user from within the MHC.

## Precondition

None.

## Parameters

None.

## Returns

None.

## Example

```c
DWT_Initialize();
```
