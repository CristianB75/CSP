MCANx_TxFIFOIsFull Function
C
bool MCANx_TxFIFOIsFull();
Preconditions

MCANx_Initialize must have been called for the associated MCAN instance.

Returns

true - Tx FIFO is full. false - Tx FIFO is not full.

Remarks

None.

Example
if (MCAN0_TxFIFOIsFull() == false)
{
    // Tx FIFO is not full
}
