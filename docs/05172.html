Library Interface
a) Initialization Function
 
Name 
Description 
 
MCANx_Initialize 
Initializes given instance of the MCAN peripheral. 
b) Setup Functions
 
Name 
Description 
 
MCANx_TxBuffersCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given MCAN's Tx transfer events occur. 
 
MCANx_MessageRAMConfigSet 
Set the Message RAM Configuration. 
 
MCANx_TxFifoCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given MCAN's Tx transfer events occur. 
 
MCANx_TxEventFifoCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given MCAN's Tx transfer events occur. 
 
MCANx_RxBuffersCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given MCAN's Rx transfer events occur. 
 
MCANx_RxFifoCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given MCAN's Rx transfer events occur. 
c) Transaction Functions
 
Name 
Description 
 
MCANx_ExtendedFilterElementGet 
Get a Extended filter element configuration. 
 
MCANx_MessageTransmit 
Transmits a message into CAN bus from the specific Tx buffer. 
 
MCANx_ExtendedFilterElementSet 
Set a Extended filter element configuration. 
 
MCANx_TxBufferIsBusy 
Check if Transmission request is pending for the specific Tx buffer. 
 
MCANx_InterruptClear 
Clears Interrupt status. 
 
MCANx_MessageTransmitFifo 
Transmit multiple messages into CAN bus from Tx FIFO. 
 
MCANx_InterruptGet 
Returns the Interrupt status. 
 
MCANx_TxFifoFreeLevelGet 
Returns Tx FIFO Free Level. 
 
MCANx_TxEventFifoRead 
Read Tx Event FIFO for the transmitted messages. 
 
MCANx_TxEventFifoFillLevelGet 
Returns Tx Event FIFO Fill Level. 
 
MCANx_MessageReceive 
Read a message from the specific Rx Buffer. 
 
MCANx_StandardFilterElementGet 
Get a standard filter element configuration. 
 
MCANx_RxBufferNumberGet 
Get Rx Buffer Number. 
 
MCANx_StandardFilterElementSet 
Set a standard filter element configuration. 
 
MCANx_MessageReceiveFifo 
Read messages from Rx FIFO0/FIFO1. 
 
MCANx_RxFifoFillLevelGet 
Returns Rx FIFO0/FIFO1 Fill Level. 
 
MCANx_SleepModeEnter 
Puts the MCAN Peripheral in sleep mode (clock stop request). 
 
MCANx_SleepModeExit 
Exits MCAN peripheral from sleep mode (clock stop request). 
d) Status Functions
 
Name 
Description 
 
MCANx_ErrorCountGet 
Returns the transmit and receive error count during transfer. 
 
MCANx_ErrorGet 
Returns the error during transfer.
