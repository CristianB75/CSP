Library Interface
a) Initialization Function
 
Name 
Description 
 
PDECx_COUNTERInitialize 
Initializes given instance of PDEC peripheral in counter mode. 
b) Setup Functions
 
Name 
Description 
 
PDECx_COUNTERCallbackRegister 
Registers the function to be called from interrupt. 
c) Transaction Functions
 
Name 
Description 
 
PDECx_COUNTERCompare0Get 
Reads Compare 0 register value 
 
PDECx_COUNTERCompare0Set 
Configures Compare 0 register value 
 
PDECx_COUNTERCompare1Get 
Reads Compare 1 register value 
 
PDECx_COUNTERCompare1Set 
Configures Compare 0 register value 
 
PDECx_COUNTERStart 
Starts the given PDEC counter in counter mode. 
 
PDECx_COUNTERStop 
Stops the given PDEC counter in counter mode. 
d) Status Functions
 
Name 
Description 
 
PDECx_COUNTERStatusGet 
Identifies status of the counter mode interrupts 
e) Data Types and Constants
 
Name 
Description 
 
PDEC_COUNTER_CALLBACK 
Defines the function pointer data type and function signature for the pdec counter callback function. 
 
PDEC_COUNTER_CALLBACK_OBJ 
Callback structure 
 
PDEC_COUNTER_STATUS 
Identifies counter mode interrupt source mask.
