Running The Application

MCU Devices

  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Connect to the EDBG Virtual COM port and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Open the Project trustZone_basic (Project Group) in the MPLAB X IDE and Set Non-secure project sam_l11_xpro_NonSecure.X as Main Project.
  4. Open trustZone_basic (Project Group) in the MPLAB X IDE. Double click on trustZone_basic_sam_l11_xpro_NonSecure to open the non-secure project
  5. Set trustZone_basic_sam_l11_xpro_NonSecure as the main project
  6. Build and program the application (trustZone_basic_sam_l11_xpro_NonSecure)
  7. Observe the output as described in the following table
Kit Name 
Output 
SAM L11 Xplained Pro Evaluation Kit 
For Add operation:

  • Press 'a' on the terminal
  • Enter first 16 bit unsigned number and press 'Enter' on keyboard
  • Enter second 16 bit unsigned number and press 'Enter' on keyboard
  • Observe the result. Result is the addition of first and second 16 bit unsigned number.

For Multiply operation:

  • Press 'b' on the terminal
  • Enter first 16 bit unsigned number and press 'Enter' on keyboard
  • Enter second 16 bit unsigned number and press 'Enter' on keyboard
  • Observe the result. Result is multiplication of first and second 16 bit unsigned number.
