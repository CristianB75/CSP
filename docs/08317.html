MPLAB Harmony Configurations
Components Used
  • EIC - Configured as Mix-Secure Peripheral for the Non-secure EIC interrupt
  • Event System - Configured as Mix-Secure Peripheral for the Secure and Non-secure events
  • RTC - Configured as Secure mode
  • LED - Configured as Non-secure mode
  • SECURE_PIN and NON_SECURE_PIN - Configured as Secure and Non-secure mode respectively

 

NOTE: For projects that use IAR Embedded work bench as IDE, the harmony project file should be opened using MHC plugin (standalone mode) to access the project graph.
