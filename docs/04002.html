MCANx_RxCallbackRegister Function
C
void MCANx_RxCallbackRegister(
    MCAN_CALLBACK callback, 
    uintptr_t contextHandle, 
    MCAN_MSG_RX_ATTRIBUTE msgAttr
);
Preconditions

MCANx_Initialize must have been called for the associated MCAN instance.

Parameters
Parameters 
Description 
callback 
A pointer to callback function which function prototype defined by the MCAN_CALLBACK data type.
 
contextHandle 
A value passed into the callback function as callback parameter.
 
msgAttr 
Message to be read from Rx FIFO0 or Rx FIFO1 or Rx Buffer 
Returns

None.

Remarks

None.

Example
void MCAN0_callbackHandler (uintptr_t contextHandle)
{
}
static uint32_t rx_messageID = 0;
static uint8_t rx_message[64] = {0};
static uint8_t rx_messageLength = 0;
static uint16_t timestamp = 0;
MCAN0_RxCallbackRegister( MCAN0_callbackHandler, 0, MCAN_MSG_ATTR_RX_FIFO0 );
MCAN0_MessageReceive(&rx_messageID, &rx_messageLength, rx_message, &timestamp, MCAN_MSG_ATTR_RX_FIFO0);
