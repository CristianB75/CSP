```c
// x - Instance of the LCDC peripheral

uint32_t LCDCx_LAYER_IRQ_Status(LCDCx_LAYER_ID layer);
```

## Summary

Reads the status of the interrupt from the specified layer

## Description

None

## Precondition

None.

## Parameters

|Param|Description|
|-----|-----------|
|layer|the layer|

## Returns

*uint32\_t* - the value of the layer's interrupt register
