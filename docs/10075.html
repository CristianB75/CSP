Library Interface
a )Initialization Function
 
Name 
Description 
 
USARTx_Initialize 
Initializes given instance of the USART peripheral. 
b) Setup Functions
 
Name 
Description 
 
USARTx_SerialSetup 
Sets the USART serial communication settings dynamically. 
 
USARTx_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given USART's read events occur. 
 
USARTx_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given USART's write events occur. 
 
USARTx_ReadCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given USART's read events occur. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_WriteCallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given USART's write events occur. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_ReadNotificationEnable 
This API lets the application turn the receive notifications on/off. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_WriteNotificationEnable 
This API lets the application turn the transmit notifications on/off. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_ReadThresholdSet 
This API allows the application to set a threshold level on the number of bytes of data available in the receive buffer. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_WriteThresholdSet 
This API allows the application to set a threshold level on the number of free space available in the transmit buffer. This API is only available when USART ring buffer mode is enabled in MHC. 
c) Transaction Functions
 
Name 
Description 
 
USARTx_Read 
Submits a read buffer to the given USART peripheral to process. 
 
USARTx_Write 
Submits a write buffer to the given USART peripheral to transfer. 
 
USARTx_ReadByte 
Submits request to read a byte of data to the given USART peripheral. 
 
USARTx_WriteByte 
Submits a byte of data to the given USART peripheral to transfer. 
 
USARTx_Read 
Read the requested data from the receive buffer into the application buffer. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_Write 
Submits a write buffer to the given USART peripheral to transfer. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_ReadAbort 
Aborts the ongoing read request. This API is available when UART ring buffer mode is disabled in MHC. 
d) Status Functions
 
Name 
Description 
 
USARTx_ErrorGet 
Gets the error of the given USART peripheral instance. 
 
USARTx_ReadCountGet 
Returns the count of number of bytes processed for a given USART read operation. 
 
USARTx_WriteCountGet 
Returns the count of number of bytes processed for a given USART write operation. 
 
USARTx_ReadIsBusy 
Returns the read request status associated with the given USART peripheral instance. 
 
USARTx_WriteIsBusy 
Returns the write request status associated with the given USART peripheral instance. 
 
USARTx_ReceiverIsReady 
Returns the hardware status of the USART Receiver. 
 
USARTx_TransmitterIsReady 
Returns the hardware status of the USART Transmit FIFO. 
 
USARTx_TransmitComplete 
Returns the hardware status of the USART Transmit Shift Register. 
 
USARTx_ReadBufferSizeGet 
Returns the size of the receive ring buffer. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_WriteBufferSizeGet 
Returns the size of the internal transmit ring buffer. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_ReadCountGet 
Returns the number of bytes available in the receive buffer. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_WriteCountGet 
Returns the number of bytes pending to be transmitted out in the transmit buffer. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_ReadFreeBufferCountGet 
Returns the number of bytes of free space available in the internal receive buffer. This API is only available when USART ring buffer mode is enabled in MHC. 
 
USARTx_WriteFreeBufferCountGet 
Returns the number of bytes of free space available in the internal transmit buffer. This API is only available when USART ring buffer mode is enabled in MHC. 
e) Data Types and Constants
 
Name 
Description 
 
USART_CALLBACK 
Defines the data type and function signature for the USART peripheral callback function. 
 
USART_RING_BUFFER_CALLBACK 
Defines the data type and function signature for the USART peripheral callback function in the ring buffer mode. 
 
USART_EVENT 
Defines the data type for the USART events. 
 
USART_DATA_BIT 
Defines the data type for the USART data bit length. 
 
USART_ERROR 
Defines the data type for the USART peripheral errors. 
 
USART_PARITY 
Defines the type of parity for USART peripheral. 
 
USART_SERIAL_SETUP 
Defines the data type for the USART serial setup. 
 
USART_STOP_BIT 
Defines the length of stop bits.
