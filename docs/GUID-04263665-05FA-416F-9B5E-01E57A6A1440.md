    /* Pin AN1 */

    /* Pin AN2 */

    AC_POSINPUT_VSCALE,

## Summary

Identifies the positive inputs of a comparator channel

## Description

This enumeration can be used as an argument to APIs where it needs to<br />indicate a positive input of a comparator.
