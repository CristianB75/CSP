Running The Application

MCU Devices

  1. Open evsys_trigger (Project Group) in the MPLAB X IDE. Double click on evsys_trigger_sam_l11_xpro_NonSecure to open the non-secure project
  2. Set evsys_trigger_sam_l11_xpro_NonSecure as the main project
  3. Build and program the application (evsys_trigger_sam_l11_xpro_NonSecure)
  4. Observe the output as described in the following table
Kit Name 
Output 
SAM L11 Xplained Pro Evaluation Kit 

  • Periodic RTC event (every ~250ms) toggles PA08 (secure pin) and PA09 (non-secure pin) in secure project.
  • LED0 is toggled on Switch (SW0) press in non-secure project
