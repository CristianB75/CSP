Library Interface
a) Initialization Function
 
Name 
Description 
 
CCPx_CompareInitialize 
Initializes given instance of CCP. 
b) Setup Functions
 
Name 
Description 
 
CCPx_CompareCallbackRegister 
Registers the function to be called from interrupt. 
c) Transaction Functions
 
Name 
Description 
 
CCPx_Compare16bitValueGet 
Reads the timer current count value. 
 
CCPx_Compare16bitValueSet 
Sets new timer counter value. 
 
CCPx_Compare32bitPeriodValueGet 
Reads the period value of given timer. 
 
CCPx_Compare16bitPeriodValueGet 
Reads the period value of given timer. 
 
CCPx_Compare16bitPeriodValueSet 
Sets the period value of a given timer. 
 
CCPx_Compare16bitRAValueGet 
Gets lower 16 bits of compare value. 
 
CCPx_Compare16bitRAValueSet 
Writes compare value of the given compare channel 0. 
 
CCPx_Compare16bitRBValueGet 
Gets higher 16 bits of compare value. 
 
CCPx_Compare16bitRBValueSet 
Writes higher 16bits of compare value. 
 
CCPx_Compare32bitPeriodValueSet 
Sets the period value of a given timer. 
 
CCPx_Compare32bitValueGet 
Reads the 32bit comapre value. 
 
CCPx_Compare32bitValueSet 
Sets 32bit compare value. 
 
CCPx_CompareAutoShutdownClear 
Clears auto shutdown event status. 
 
CCPx_CompareAutoShutdownSet 
Enable auto shutdown. 
 
CCPx_CompareStart 
Starts the timer for given CCP instance. 
 
CCPx_CompareStop 
Stops the timer for given CCP instance. 
e) Data Types and Constants
 
Name 
Description 
 
CCPx_COMPARE_CALLBACK 
Defines the function pointer data type and function signature for the ccp channel callback function.
