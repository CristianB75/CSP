Microchip Support
Library Interface
a) Initialization Function
 
Name 
Description 
 
SMC_Initialize 
Initializes Static Memory Controller (SMC) interface. 
b) Setup Function
 
Name 
Description 
 
PMECC_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given PMECC's interrupt occurred. 
 
PMERRLOC_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given PMERRLOC's interrupt occurred. 
 
SMC_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the given SMC's interrupt occurred. 
c) Transaction Function
 
Name 
Description 
 
SMC_AddressWrite 
Writes address to the External Bus Interface (EBI) address. 
 
SMC_AddressWrite16 
Writes 16-bit address to the External Bus Interface (EBI) address. 
 
SMC_CommandWrite 
Writes command to the External Bus Interface (EBI) address. 
 
SMC_CommandWrite16 
Writes 16-bit command to the External Bus Interface (EBI) address. 
 
SMC_DataAddressGet 
Returns the External Bus Interface (EBI) address for the requested chip select number. 
 
SMC_DataRead 
Reads data from the External Bus Interface (EBI) address. 
 
SMC_DataRead16 
Reads 16-bit data from the External Bus Interface (EBI) address. 
 
SMC_DataWrite 
Writes data to the External Bus Interface (EBI) address. 
 
SMC_DataWrite16 
Writes 16-bit data to the External Bus Interface (EBI) address. 
 
PMECC_DataPhaseStart 
Enables PMECC Write access and start data phase. 
 
PMECC_ECCGet 
Returns error correction code (ECC). 
 
PMECC_ErrorGet 
Returns PMECC error. 
 
PMECC_RemainderGet 
Returns PMECC remainder. 
 
PMECC_StatusIsBusy 
Returns PMECC busy status. 
 
PMERRLOC_ErrorLocationDisable 
Disables error location of memory. 
 
PMERRLOC_ErrorLocationFindNumOfRoots 
Find the number of roots using PMERRLOC acceleration. 
 
PMERRLOC_ErrorLocationGet 
Find and returns error location of memory. 
 
PMERRLOC_SigmaSet 
Set sigma value to the PMERRLOC register. 
d) Data Types and Constants
 
Name 
Description 
 
PMECC_CALLBACK 
Defines the data type and function signature for the PMECC peripheral callback function. 
 
PMERRLOC_CALLBACK 
Defines the data type and function signature for the PMERRLOC peripheral callback function. 
 
SMC_CALLBACK 
Defines the data type and function signature for the SMC peripheral callback function.
