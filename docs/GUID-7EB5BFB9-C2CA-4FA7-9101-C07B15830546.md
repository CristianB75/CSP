```c
void CCL_Initialize (void)
```

## Summary

Initializes CCL peripheral.

## Description

This function initializes the CCL PLIB as configured in MCC.

## Preconditions

MCC GUI should be configured with the right values.

## Returns

None.

## Example

```c
CCL_Initialize();
```
