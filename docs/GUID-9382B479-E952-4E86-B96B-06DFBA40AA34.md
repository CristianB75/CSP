{
    SINGLE_MODE = 0x0,
    DUAL_MODE = 0x1,
    QUAD_MODE = 0x2
} SQI_LANE_MODE;

## Summary

Defines the data type to specify the type of lane.

## Description

SQI Lane Mode can be used to specify the lane to be used for SQI transaction
