Running The Application

MPU Devices

  1. Open the Terminal application (Ex.:Tera term) on the computer.
  2. Select the JLink CDC UART Port in Terminal application (Ex.:Tera term) and configure the serial settings as follows:
    • Baud : 115200
    • Data : 8 Bits
    • Parity : None
    • Stop : 1 Bit
    • Flow Control : None
  3. Build the MPLAB X project using MPLAB X IDE
  4. Use the output binary file (harmony.bin) to setup the board as detailed in hardware setup
  5. Once COM port connection is established, you should see the following message in the console. The console displays the initial transmit and receive buffer sizes. It then adds 10 characters - "0123456789" to the transmit buffer and displays the available free space in the transmit buffer after the 10 characters have been added. After this, the message asks the user to enter 10 characters. After all the 10 characters are received, it will echo it back on the terminal.

(Note: For this example, the "Free Space in Transmit Buffer = xx", where xx can be anything between 117 to 127, depending on the DBGU baud rate and the CPU execution speed). 

 

 

  1. Type 10 characters in the terminal. It will echo back the received bytes and then print a message indicating that the receive notification is turned on.

 

  1. Again type 10 characters in the terminal. After every 10 characters are received, it will echo it back on the terminal.
