Library Interface

This section describes the Application Programming Interface (API) functions of the <library_name> Library. 

Refer to each section for a detailed description.

a) Initialization Function
 
Name 
Description 
 
NVMCTRL_Initialize 
Initializes given instance of the NVMCTRL peripheral. 
b) Setup Functions
 
Name 
Description 
 
NVMCTRL_CallbackRegister 
Sets the pointer to the function (and it's context) to be called when the operation is complete. 
c) Transaction Functions
 
Name 
Description 
 
NVMCTRL_RowErase 
Erases a Row in the NVM 
 
NVMCTRL_Read 
Reads length number of bytes from a given address in FLASH memory. 
 
NVMCTRL_RWWEEPROM_RowErase 
Erases a Row in the RWWEEPROM. 
 
NVMCTRL_CacheInvalidate 
Invalidates all cache lines. 
 
NVMCTRL_RegionLock 
Locks a NVMCTRL region. 
 
NVMCTRL_RegionUnlock 
Unlocks a NVM region. 
 
NVMCTRL_PageWrite 
Writes one page of data to given NVM address. 
 
NVMCTRL_RWWEEPROM_PageWrite 
Writes one page of data to given RWWEEPROM address. 
 
NVMCTRL_RWWEEPROM_Read 
Reads length number of bytes from a given address in Data FLASH memory. 
 
NVMCTRL_PageBufferWrite 
Writes data to the internal buffer of NVM known as the page buffer 
 
NVMCTRL_PageBufferCommit 
Commits the data present in NVM internal page buffer to flash memory 
d) Data Types and Constants
 
Name 
Description 
 
NVMCTRL_CALLBACK 
Defines the data type and function signature for the NVMCTRL peripheral callback function. 
 
NVMCTRL_ERROR 
Defines the NVMCTRL Error Type. 
 
NVMCTRL_FLASH_START_ADDRESS 
Defines the start address of NVMCTRL Flash. 
 
NVMCTRL_FLASH_PAGESIZE 
Defines the size (in bytes) of a NVMCTRL Page. 
 
NVMCTRL_FLASH_ROWSIZE 
Defines the size (in bytes) of a NVMCTRL Row. 
 
NVMCTRL_FLASH_SIZE 
Defines the size (in bytes) of Flash. 
 
NVMCTRL_RWWEEPROM_START_ADDRESS 
Defines the start address of NVMCTRL RWWEEPROM. 
 
NVMCTRL_RWWEEPROM_PAGESIZE 
Defines the size (in bytes) of a NVMCTRL RWWEEPROM Page. 
 
NVMCTRL_RWWEEPROM_ROWSIZE 
Defines the size (in bytes) of a NVMCTRL RWWEEPROM Row. 
 
NVMCTRL_RWWEEPROM_SIZE 
Defines the size (in bytes) of RWWEEPROM. 
d) Status Functions
 
Name 
Description 
 
NVMCTRL_IsBusy 
Returns the current status of NVM controller. 
 
NVMCTRL_ErrorGet 
Returns the error state of NVM controller.
