Library Interface
a) Common Function
 
Name 
Description 
 
POWER_LowPowerModeEnter 
Puts the device in selected low power mode 
b) Deep Sleep Functions
 
Name 
Description 
 
POWER_Initialize 
Initializes Deep Sleep controller of Power module. 
 
POWER_DSGPR_Read 
Reads saved data 
 
POWER_DSGPR_Write 
Saves critical data in deep sleep state 
 
POWER_ReleaseGPIO 
Release GPIO pins 
 
POWER_WakeupSourceClear 
Clears the wakeup source 
 
POWER_WakeupSourceGet 
Returns the source of wakeup from deep sleep 
c) Data Types and Constants
 
Name 
Description 
 
POWER_DSGPR 
Identifies deep sleep GPRs 
 
POWER_LOW_POWER_MODE 
Identifies low power mode options 
 
POWER_WAKEUP_SOURCE 
Identifies wake up sources from deep sleep mode
