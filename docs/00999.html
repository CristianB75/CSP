Library Interface
a) Initialization Function
 
Name 
Description 
 
CANx_Initialize 
Initializes given instance of the CAN peripheral. 
b) Setup Functions
 
Name 
Description 
 
CANx_RxCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given CAN's Rx transfer events occur. 
 
CANx_TxCallbackRegister 
Sets the pointer to the function (and it's contextHandle) to be called when the given CAN's Tx transfer events occur. 
c) Transaction Functions
 
Name 
Description 
 
CANx_ErrorGet 
Returns the error during transfer. 
 
CANx_InterruptClear 
Clears Interrupt status. 
 
CANx_InterruptGet 
Returns the Interrupt status. 
 
CANx_MessageRAMConfigSet 
Set the Message RAM Configuration. 
 
CANx_MessageTransmit 
Transmits a message into CAN bus. 
 
CANx_ExtendedFilterElementGet 
Get a Extended filter element configuration. 
 
CANx_ExtendedFilterElementSet 
Set a Extended filter element configuration. 
 
CANx_MessageReceive 
Receives a message from CAN bus. 
 
CANx_StandardFilterElementGet 
Get a standard filter element configuration. 
 
CANx_StandardFilterElementSet 
Set a standard filter element configuration. 
 
CANx_TransmitEventFIFOElementGet 
Get the Transmit Event FIFO Element for the transmitted message. 
d) Status Functions
 
Name 
Description 
 
CANx_TxFIFOIsFull 
Returns true if Tx FIFO is full otherwise false. 
 
CANx_ErrorCountGet 
Returns the transmit and receive error count during transfer. 
e) Data Types and Constants
 
Name 
Description 
 
CAN_CALLBACK 
CAN Callback Function Pointer. 
 
CAN_ERROR 
CAN Transfer Error data type. 
 
CAN_INTERRUPT_MASK 
CAN Interrupt Mask. 
 
CAN_MODE 
CAN Mode for Classic CAN and CAN FD. 
 
CAN_MSG_RAM_CONFIG 
CAN Message RAM Configuration structure. 
 
CAN_MSG_RX_ATTRIBUTE 
CAN Rx Message Attribute for Rx Buffer/FIFO0/FIFO1. 
 
CAN_MSG_TX_ATTRIBUTE 
CAN Tx Message Attribute for Tx Buffer/FIFO. 
 
CAN_MSG_RX_FRAME_ATTRIBUTE 
CAN Message RX Frame Attribute for Data Frame and Remote Frame. 
 
CAN_RX_MSG 
CAN RX Message Buffer structure.
